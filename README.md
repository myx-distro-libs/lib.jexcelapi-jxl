# lib.jexcelapi-jxl

AE3 packaging wrapper for **JExcelAPI (jXL)** — a Java library for reading and writing Microsoft Excel spreadsheets.

Provides: `ae3.pkg.lib.jxl`

## Bundled library

- **JExcelAPI 2.6.12** by Andy Khan
  - Homepage: http://jexcelapi.sourceforge.net
  - License: GNU Lesser General Public License v2 (LGPLv2)

This repository contains only pre-built jar files and AE3 build metadata.
No source code from the above library is included or modified.
